여백 및 간격 관련 스타일

기준 div

기준 div

padding

내용물 영역(content)과 테두리 영역(border) 사이의 거리 (안쪽 여백) 을 조정하는 스타일 속성

컨텐츠 영역
padding : 100px;

컨텐츠 영역
padding-top : 50px, padding-left : 100px;

컨텐츠 영역
padding: 50px 20px 10px 0px;

margin

타 요소들 간의 간격 (바깥여백) 을 조정할 때 사용

컨텐츠 영역
margin : 50px;

컨텐츠 영역
margin-top : 50px; margin-left : 20px

컨텐츠 영역
margin : 50px 30px 20px 0px;
