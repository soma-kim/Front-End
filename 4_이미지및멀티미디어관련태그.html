이미지 관련 태그

*scr 속성, alt 속성



* alt
- 사진의 경로가 잘못되거나 이미지를 제대로 표현할 수 없을 때 대체 텍스트의 용도
- 시각장애인들을 위한 스크린리더 (화면낭독기)에서 이미지를 읽어 주는 설명 문구 : 접근성을 높여 주는 용도로 쓰임

width 속성, height 속성

이미지의 가로, 세로길이를 조정 가능한 속성
px: 고정 길이(화면의 사이즈가 변해도 이미지의 사이즈는 변하지 않음)
% : 가변 길이(화면의 사이즈가 변하면 이미지의 사이즈도 변함)

고정 길이 단위(px): 화면 사이즈가 조정되더라도 이미지의 크기는 변동이 없음 (기본값)
